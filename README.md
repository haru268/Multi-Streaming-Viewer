# マルチ配信ビューア

YouTube、Twitch、ツイキャスのライブ配信を同時に視聴できるWebアプリケーションです。

## 機能

- **マルチストリーム視聴**: 最大16枠まで同時に配信を視聴可能
- **初期9枠**: カスタマイズされた9枠のスロットを表示
- **プラットフォーム対応**: YouTube、Twitch、ツイキャスに対応
- **チーム名表示**: 各ストリームにチーム名を設定可能
- **ローカルストレージ保存**: 設定をブラウザに自動保存
- **レイアウト調整**: 2列、3列、4列のグリッドレイアウトに対応
- **メイン表示**: 特定のストリームを強調表示
- **ミュート機能**: 各ストリームを個別にミュート可能（規約遵守: 初期状態は全枠ミュート）

## セットアップ

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev

# ビルド
npm run build
```

## 使い方

1. プラットフォーム（YouTube/Twitch/ツイキャス）を選択
2. URLまたはIDを入力
3. チーム名（オプション）を入力
4. 「適用」ボタンをクリックしてストリームを読み込む
5. 「枠を追加」ボタンで最大16枠まで追加可能
6. 「設定を保存」で現在の構成を保存
7. 各ストリームの🔊ボタンをクリックして音声をON/OFF（トグル式）

## 利用規約遵守について

本アプリケーションは、各プラットフォーム（YouTube、Twitch、ツイキャス）の利用規約を遵守するよう設計されています：

- **全枠ミュートで自動再生**: 新規追加されたストリームは必ずミュート状態で開始されます
- **ユーザー操作のみで音声ON**: 音声はユーザーが明示的にクリックした場合のみONになります
- **埋め込みプレイヤーの改変なし**: 公式の埋め込みプレイヤーをそのまま使用し、改変は行いません
- **同期再生や強制制御なし**: 複数プレイヤーの同期再生や強制制御は行いません

**注意**: ツイキャスは埋め込みパラメータでミュート制御ができないため、プレイヤー内のボタンで音声を制御してください。
